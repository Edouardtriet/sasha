🤓 Programming Fun Facts

The first computer bug was actually a real bug! In 1947, Grace Hopper found a moth causing problems in the Harvard Mark II computer. This is where the term 'debugging' comes from! 🐛

The programming language Python isn't named after a snake - it's named after the comedy group Monty Python! 🐍

💡 Did You Know?
